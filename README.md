## Auth

```
secman auth -h
secman auth login
secman auth logout
secman auth refresh
secman auth status
```

## Repo

```
secman repo -h
secman repo clone
secman repo create
secman repo fork
secman repo list
```

## Config

```
secman gh-config -h
secman gh-config get ...
secman gh-config set ...
```
